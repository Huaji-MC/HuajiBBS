# HuajiBBS 使用须知

---

### 一、简介

HuajiBBS 是由 滑稽MC 开发的一款 PHP 论坛系统，HuajiBBS致力于为用户提供最优质的服务！
官网链接：[HuajiBBS官网](https://HuajiMC.gitee.io/HuajiBBS)

### 二、下载安装包

你可以在 https://github.com/Huaji-MC/HuajiBBS/releases/ 下载对应版本安装包

### 三、安装方法

1. 上传对应版本安装包（下载方法见第二款）至服务器对应网站根目录
2. 解压安装包
3. 访问网站执行安装程序
4. 填写相关信息并安装
